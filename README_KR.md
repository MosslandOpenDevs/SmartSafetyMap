# 스마트안심맵

스마트안심맵은 건물 주변의 안전 관련 공공정보를 주제별로 선택해 직관적으로 확인할 수 있는 지도 기반 웹 서비스이다.

## 서비스 개요

건물 주변 안전정보를 지도에서 확인하는 웹 서비스

## 핵심 방식

버튼 선택 시 해당 정보만 지도에 표시

## 주요 정보

교통약자, 침수, 위험지역, 신고, 대기질, CCTV·비상벨, 대피소·병원

## 대기질 표시

좋음, 보통, 나쁨, 매우나쁨

## 이용 방식

지도 확인 → 버튼 선택 → 정보 표시

## 서비스 특징

건물 주변 정보 중심  
선택형 지도 구조  
직관적 시각화  
공공 API 연계 가능

## 상세 기획

### 서비스 목적

스마트안심맵은 건물 주변의 다양한 안전 관련 정보를 한 화면에서 확인할 수 있도록 구성한 지도 기반 웹 서비스이다.  
사용자는 필요한 주제를 선택하여 관련 정보만 볼 수 있으며, 복잡한 데이터 대신 직관적인 시각 정보로 주변 상황을 빠르게 파악할 수 있다.

### 핵심 아이디어

각 주제를 버튼 형태로 구성하고, 사용자가 원하는 버튼을 누르면 해당 카테고리 정보만 지도에 표시한다.

예시  
교통약자 버튼 선택 시 교통약자 관련 정보 표시  
침수 버튼 선택 시 침수 위험 정보 표시  
위험지역 버튼 선택 시 위험 관련 정보 표시  
신고 버튼 선택 시 사용자 신고 위치 표시  
대기질 버튼 선택 시 대기질 상태 표시

### 주요 기능

#### 교통약자
장애인 편의시설, 이동 가능 경로, 엘리베이터, 장애인 화장실, 경사로, 접근 가능한 출입구 정보 표시

#### 침수
침수 이력 구역, 침수 위험 구역 등 침수 관련 정보 표시

#### 위험지역
신고 누적 또는 공공데이터를 기반으로 주의가 필요한 구역 표시  
초기 단계에서는 범위를 넓게 잡기보다 침수 취약 구역, 신고 다발 지역처럼 세분화하는 것이 현실적임

#### 신고
사용자가 지도에서 위험 위치를 선택하고 내용과 사진을 등록하는 자체 신고 기능  
일반 사용자와 관리자가 직접 제보 가능

#### 대기질
건물 주변 대기질 정보를 좋음, 보통, 나쁨, 매우나쁨 단계로 단순 표시

#### CCTV·비상벨
건물 주변 CCTV 및 비상벨 위치 정보 표시

#### 대피소·병원
주변 대피소, 병원, 응급 대응 관련 위치 정보 표시

### 신고 기능 흐름

1. 사용자가 지도에서 위치 선택
2. 위험 내용 입력
3. 사진 업로드
4. 신고 등록
5. 지도에 해당 위치 표시
6. 신고 누적 시 위험지역 분석에 활용 가능

### 전국 맵 운영 방향

전국 단위 맵 자체는 가능하지만, 모든 항목을 전국 단위로 균일하게 제공하는 것은 어려울 수 있다.  
따라서 전국 맵을 기본으로 하되, 데이터가 없는 지역은 미제공 지역 또는 지원 예정으로 명확하게 표시하는 방식이 적절하다.

예시 문구  
해당 지역 데이터 미제공  
현재 지원하지 않는 지역입니다  
해당 항목은 일부 지역만 제공됩니다  
지역별 제공 범위가 상이합니다

## 기술적 구현 난이도

| 구분 | 항목 | 이유 |
|---|---|---|
| 쉬움 | 대기질 | 공공 API가 비교적 정리되어 있고 등급형 표시가 쉬움 |
| 쉬움 | 교통약자 | 위치 기반 시설 정보를 지도에 표시하기 쉬움 |
| 쉬움 | 대피소·병원 | 위치 정보 중심이라 구조가 단순함 |
| 쉬움 | 자체 신고 | 위치 선택, 내용 입력, 사진 업로드 방식으로 구현 가능 |
| 중간 | 침수 | 데이터는 있으나 이력·위험구역 등 성격 구분이 필요함 |
| 중간 | CCTV·비상벨 | 기능은 단순하지만 지역별 데이터 차이가 있음 |
| 중간 | 전국 맵 운영 | 지역별 미제공 데이터 처리 설계가 필요함 |
| 어려움 | 위험지역 | 위험 기준을 직접 정해야 해서 판단 로직이 필요함 |
